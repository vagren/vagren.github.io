WebCourse
  • Ubuntu
    • 最流行的发行版本
    • 每六个月发布新版本（正式版），每两年发布长期支持版本，长期支持版可获得长达5年的支持
  • LinuxMint
    • 最友好的Linux发行版，它的界面和操作系统和Windows非常类似
    • 基于Ubuntu开发的
    • 允许用户选择不同的桌面环境
  • Debian
  • Fedora
    • ZDNET认为Fedora是程序员的首选Linux, Linux Torvalds也使用Fedora进行开发工作。
    • 容易安装和设置
    • 拥有出色的开发人员门户，开发人员可以迅速查找遇到的问题答案
    • Fedora携带支持Java、C++、PHP等语言的集成开发环境
    • 主要使用Gnome桌面环境，Gnome桌面是一款创新性的桌面，集中在工作上以提高生产力
    • 是在自由软件方面最坚定的实践者，在官方的软件商店中只有自由与开源软件
  • openSuse
    • 企业级的稳定性和独特的系统工具-Yast
    • 不易上手
  • Arch Linux
    • Arch Linux使用简单
    • 以保持简单为原则，从而让用户可以更加充分的管理电脑
    • Arch的用户软件仓库（AUR）
    • Arch安装好后是没有图形界面的，没有桌面环境的
  • Manjaro Linux
    • Manjaro Linux 基于Arch Linux 的发行版拥有更良好的桌面支持和使用体验
    • 支持安装多个内核并能实现快速切换，对于测试内核及其方便
  • Antergos（停止更新）
  • Gentoo/Sabayon Linux
    • Gentoo是一个极简的Linux发行版，需要完全从源代码开始构建
    • Sabayon基于Gentoo，提供了图形化支持。
  • Kali linux
    • 包含了大量的数字化取证和渗透测试软件，是安全专家和黑客首选的Linux发行版
  • SystemRescue
    • 可用于修复分区损坏
    • 提供了数据恢复工具

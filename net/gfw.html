GFW #

GFW，Great Fire Wall，防火长城，墙。

08年奥运运前期，恐怖组织在Facebook上策划、协调并实施了对我国西部地区的一个恐怖袭击。中国政府当时要求Facebook配合审查，以防再次出现此类事件。但是Facebook以尊重保护用户隐私这样一个理由拒绝了。同时，Google也以同样的理由拒绝了中国政府的要求。大陆政府就不让不愿意配合的外国公司在中国展开业务。于是Facebook还有Google的一系列服务：搜索引擎、YouTube、Gmail、Twitter、Tumblr。

网络审查制度

经过一段时间的审查制度和GFW的越来越规范化，导致甚至一些本不涉及到政治和国民安全的互联网服务。

GFW的运作原理 #

从本地计算机发出一个数据包(Packet)请求，经过本地计算机网络计入骨干网，经由DNS解析出真实IP地址，到达国际出口，最后接入对应的服务器。

从本地计算机发出一个数据包(Packet)请求，经过本地计算机网络计入骨干网，经由DNS解析。

GFW封装互联网访问的其中一种手段：污染DNS。你的数据包发送的方式是HTTP是一种明文的流量。GFW可以探测到流量的部分内容，于是解析出一个错误的服务器地址。

VPN翻墙的实现方式以及弊端 #

Shadowsocks/SSR/V2ray #

Shadowsocks把代理服务拆分成本地和远程，实现GFW的流量全部加密。

SSR,V2ray都是类似基于类似的原理。

翻墙方式的几种分类 #

  • 软件翻墙
    • VPN（VirtualPrivateNetwork），流量加密，墙无法探测。
    • Shadowsocks以及衍生品
  • 硬件翻墙（路由器翻墙）
  • 网关模式翻墙

各种线路详解 #

  • BGP GCP
  • AWS Azure
  • PCCW IPLC

#

  • 163骨干网

  • CN2: Chinatelecom Next Carrier Network, CNCN，中国电信下一代承载网络

    • 缺点：价格非常高，个人用户很少
    • CN2线路包括两种类型：CN2GT、CN2GIA
      • CN2GT: Global Transit
      • CN2GIA: Global Internet Access

  • QoS: Quality of Service, 服务质量。QoS能够识别网络中的流量需求，需要延迟低的服务优先通行；

  • BGP协议：Border Gateway Protocol.边界网关协议。

    • BGP机房一般叫多线机房
    • BGP可以自动之别你的线路是电信、移动还是联通，并且自动使用最适合你连接服务器的线路。而且在你当前网络不通的时候还会自动切换到其他的线路来连接服务器。

  • GCP: Google Cloud Platform

  • AWS: Amazon Web Service

  • Azure：微软

  • PCCW: 香港电讯盈科提供的线路

  • IPLC： Internationnal Private Leased Circuit
